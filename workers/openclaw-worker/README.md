# OpenClaw Worker

This worker delegates tasks to an OpenClaw runtime.

## Setup
- Ensure OpenClaw is running and reachable
- Provide OpenClaw session routing if needed

## Env
- `OPENCLAW_URL` (default: http://localhost:3001)
- `OPENCLAW_API_KEY` (optional)

## Notes
This is a stub integration. It will send task prompts to OpenClaw and return summaries.
