# Contributing to Clawpany

Thanks for helping build Clawpany. This project is open, but we keep changes tight and auditable.

## Quick start
1) Pick an issue: https://github.com/chocodee/clawpany/issues
2) Comment to claim it
3) Create a branch + PR

## Expectations
- Small, reviewable changes
- Prefer tests for new behavior
- Follow rustfmt + clippy (CI enforces)
- No large refactors without discussion

## Local dev
```bash
cargo fmt --all
cargo clippy --workspace --all-targets -- -D warnings
cargo test --workspace
```

## PR checklist
- [ ] Clear scope + description
- [ ] Tests added/updated if relevant
- [ ] CI passes

## Support
- Open a GitHub issue or comment on your PR
- Or DM the project owner

## Payment
Some issues are funded by real client work.
Approved contributions can receive token allocations or direct payment.
