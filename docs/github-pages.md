# GitHub Pages Setup

## Option A: Pages from /docs
1. GitHub → Repo Settings → Pages
2. Source: `main` branch, `/docs` folder
3. Save

Your site will be published at:
`https://chocodee.github.io/clawpany/`

## Option B: Pages from / (root)
Place a static site (index.html) in repo root and select `/ (root)`.

---

This repo currently uses markdown docs, so **/docs** is the easiest.
